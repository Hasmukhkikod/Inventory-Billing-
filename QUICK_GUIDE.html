Quick Start Guide for Business Users

Getting Started

1

Start the Software

Double-click "START.bat" file. A black window opens — keep it open. Your browser will open automatically.

2

Login

Enter your email and password on the login page. Contact your admin if you don't have credentials.

3

You're In!

The Dashboard opens with your sales summary, charts, and quick action buttons. Use the sidebar menu to navigate.

Important: Always keep the black command window open while using the software. To stop, just close that window.

First Time Setup (Do This First)

1

Settings → Company Details

Add your business name, logo, GSTIN, phone, email, and address. This appears on all invoices.

2

Settings → Invoice & Tax

Set your invoice prefix (e.g. INV-), GST slabs (click "+" to add rates like 5%, 12%, 18%, 28%), and your state code for CGST/SGST.

3

Settings → Bank Details

Add bank account details and UPI ID. These appear on printed invoices for customers to pay.

4

Add Products

Go to Inventory → Add New Product. Add product name, SKU, HSN code, cost price, selling price, GST%, and opening stock. Use the "+" button to quickly add new categories, brands, or units.

5

Add Customers & Suppliers

Add your regular customers and suppliers. You can also quickly add them from the billing/purchase screens using the "+" button.

Your Daily Workflow

Morning

1. Double-click START.bat → Login

2. Check Dashboard for yesterday's summary

3. Check low stock alerts → Create purchase orders if needed

▼

During Business Hours

1. Open POS & Billing → Create invoices for walk-in / regular customers

2. Search product or scan barcode → Add to cart → Select payment → Generate Invoice

3. Use Hold Bill (F3) if customer needs to step away, Recall (F5) when they return

4. Handle Returns if any customer returns goods

5. Log any Expenses (rent, supplies, etc.)

▼

End of Day

1. Go to POS & Billing → Day-End Report

2. Review: total sales, payment mode breakdown, cash position

3. Print the report for your records

4. Go to Settings → Database Backups → Create Backup

5. Close the black command window to stop the software

Useful Tips

Quick Add: Every dropdown has a "+" button. You can add new customers, suppliers, categories, brands, units, and expense categories without leaving the current form.
Coupons: Go to Settings → Coupons tab to create discount codes. Cashiers can type the code at checkout to apply discounts automatically.
GST Auto-Calculation: Set your state code in Settings. If a customer is from the same state → CGST + SGST. Different state → IGST. Walk-in customers default to same state.
Split Payment: At billing checkout, click "Split" to pay with multiple methods. Example: ₹500 Cash + ₹300 UPI + ₹200 Card.
Barcode Labels: Open any product → Click "Generate Barcode" → Print labels for your products.
Backup: Go to Settings → Create Backup regularly. This saves your entire database. You can download the backup file to keep it safe.
Data Safety: Your data is stored locally on this computer. If you format the PC or delete the software folder, your data is gone. Always keep backups!

Troubleshooting

Software won't start?

Make sure you're double-clicking START.bat (not any other file). Keep the black window open.

Browser shows blank page?

Wait 5 seconds and refresh. Or manually go to http://localhost:8000

Forgot password?

Contact your admin. They can change your password from Users module.

"Windows protected your PC" popup?

Click "More info" → Click "Run anyway". This is normal for new software.
