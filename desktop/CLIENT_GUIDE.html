Extract the ZIP File

Right-click on Grovixo-IIMS-Portable-v2.0.zip and select "Extract All". Choose any folder (Desktop, D:\Software, or anywhere you prefer).

⚠️ Important: Do NOT extract inside Program Files folder. Use Desktop or any other folder.
▼
2
▶️

Double-Click "Start Grovixo.bat"

Open the extracted folder and double-click Start Grovixo.bat. A black command window will open — keep it open while using the software.

⚠️ Troubleshooting

"Windows protected your PC" popup? → Click "More info" → Click "Run anyway". This is normal for new software.

Browser doesn't open? → Wait 5 seconds, then manually open http://localhost:8000

Port already in use? → Close any other billing/server software, then try again.
